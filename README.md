               NUTRISSNAP AI - README
=====================================================

WHAT IS THIS?
NutriSnapAI is a smart app that lets you take a photo of your 
food and instantly get a nutritional breakdown. 

It tells you:
- What the food is.
- Estimated Calories, Protein, Carbs, and Fats.
- A Health Score (0-100).
- A brief AI summary of how healthy the meal is.

-----------------------------------------------------
HOW TO START THE APP
-----------------------------------------------------
To install the necessary files and start the app:

1. Locate the file named: install and run.bat
2. Double-click it.

The script will automatically install the requirements 
(Streamlit, OpenAI, Pillow) and launch the app in 
your web browser.

-----------------------------------------------------
ONLINE VERSION
-----------------------------------------------------
